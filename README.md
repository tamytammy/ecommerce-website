# jQuery E-commerce Website

這原本是一個jQuery製作的電商網站，用來模擬線上購物的商品瀏覽、購物車互動功能。

---

## 預期功能介紹

- 商品列表展示（jQuery 配合JSON檔案 動態渲染）
- 商品加入購物車（可即時增加數量）
- 移除商品、計算總金額
- 購物車 icon 數量更新
- 會員註冊與登入(LocalStorage作為測試環境)

---

## 使用技術

- HTML5 / CSS3
- JavaScript / jQuery
- jQuery Ajax（若有模擬串接 API）
- localStorage（模擬儲存購物車資料）
